# Cloud Storage with Netty Framework

## Для запуска необходимо:
1. Скачать гит-репозиторий проекта. Для этого вводим следующую команду в терминал:<br>
    `git clone https://github.com/central-university-dev/cloud-storage-part-1-hw-3-m1hail1337.git`
2. Запустить [скрипт Gradle](build.gradle).
3. Запустить [сервер](server/src/main/java/ru/tinkoff/semenov/ServerApp.java)
4. Запустить [клиент](client/src/main/java/ru/tinkoff/semenov/Main.java)
